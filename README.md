## Installation

Install prerequisites:
- curl
- git
- python3
- zsh
- vscode
- tmux

```bash
git clone git@github.com:pasierb/dotfiles.git
python3 ./bootstrap.py
```

### tmux

In tmux type `C-a I` to install plugins

### vim

In vim type `:PlugInstall` to install plugins
